What it does

A focused desktop client for YouTube Music — stream, organize, and play without a browser tab.

Enhanced search
Filter by songs, artists, albums, playlists, or podcasts. Top-result hero, recent queries, and suggestions from your listening history.
Playback
Queue, shuffle, repeat, crossfade, gapless handoff, global media keys, and resume where you stopped.
Library
Liked songs, playlists, history, smart playlists, and import from public YouTube playlists.
Lyrics
Synced lyrics in Now Playing, with line, phrase, or word highlighting and per-track timing offset.
Downloads
Save tracks as MP3 and play them from the app when you are offline.
Desktop
Mini player, keyboard shortcuts, system media keys, and optional auto-update via GitHub releases.
Discord
Show what you're playing on your Discord profile — track, artist, artwork, and optional playback progress. Enable it in Settings.
Appearance
Dark, glass, and light themes — frosted panels and ambient color like Apple Music.

Release history as a mind map — click a version or topic to read what changed.

Download

Get the latest PMusic-Setup-*.msi from GitHub Releases. The installer is built for 64-bit Windows 10 and later.

  • 64-bit Windows 10 or 11
  • Internet for streaming
  • YouTube / Google account optional (for library sync)
